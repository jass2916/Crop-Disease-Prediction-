# 🌾 AgroPulse: Deep Learning for Precision Agriculture

AgroPulse is an end-to-end AI solution designed to diagnose plant diseases and pest infestations across a wide variety of crops. By utilizing a deep learning architecture, the model can identify 147 distinct categories, ranging from fungal rusts in wheat to pest attacks in cotton.


# 🚀 Features

• High-Cardinality Classification: Supports 147 classes including diseases, pests, and healthy states.

• Multi-Crop Diagnostics: Analyzes health for Potato, Wheat, Rice, Cotton, Sugarcane, Corn, and various fruits.

• Real-Time Inference: Integrated with a Flask backend for seamless image processing and prediction.

• Pest Detection: Specialized identification for Aphids, Bollworms, Stem flies, and Mites.


# 🛠️ Tech Stack

• Languages: Python

• Libraries: TensorFlow/Keras, Scikit-Learn, OpenCV

• Backend: Flask

• Tools: Linux Administration, SQL, GitHub


# 📂 Project Structure

• app.py: The Flask server handling model requests.

• class_indices.json: Mapping of 147 model indices to human-readable labels.

• models/: Pre-trained deep learning models for plant health analysis.


# 📊 Model Capability

The model covers a diverse range of agricultural issues, such as:

• Wheat: Brown leaf rust, Stem fly, and Yellow rust.

• Cotton: American Bollworm, Anthracnose, and Bacterial Blight.

• Rice: Brown Spot, Leaf Blast, and Neck Blast.

• Potato: Early Blight and Late Blight.
